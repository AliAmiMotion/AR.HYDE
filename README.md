# Ari Hyde Social Resume Site

## Files

- `index.html` — site content
- `styles.css` — responsive styling
- `script.js` — scroll reveals and animated statistics

## Before publishing

1. Replace `your@email.com` in `index.html` with your real email.
2. The uploaded portrait is already included as `ari-hyde-portrait.jpg`.
3. Upload every file in this folder to the root of the GitHub repository.
4. Vercel will redeploy automatically after the commit.

## Local preview

Open `index.html` in a browser.
